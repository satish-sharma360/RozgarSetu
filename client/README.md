# RozgarSetu Client

Vite + React + TypeScript + Tailwind scaffold.

Install and run:

```bash
cd client
npm install
npm run dev
```

Default dev server: http://localhost:5173 (or the port Vite shows).

Notes:
- You must install dependencies after scaffolding.
- If Tailwind CSS classes don't apply, ensure PostCSS/Tailwind are installed and `index.css` is imported (already done in `main.tsx`).
